Souvent on veut exécuter une action plusieurs fois et/ou en plusieurs endroits d'un script.
Les fonctions permettent d'éviter les répétitions.
Les fonctions permettent aussi de corriger un bug en un seul endroit, de changer l'implémentation interne sans changer les appels
et enfin de réutiliser une portion de code dans d'autres scripts ou projets...

Vous connaissez déjà des fonctions : alert, prompt, confirm et write... : 

                Syntaxe :

                function nomDeLaFonction () {
                    ...Corps de la fonction...
                    ...Code à exécuter...
                }

                function nomDeLaFonction (param1, param2..., paramN) {
                    ...Corps de la fonction...
                    ...Code à exécuter...
                }

                l'appel (l'exécution) se fait avec le nom de la fonction et les parenthèses.

Exemple:

        

        function maFonction() {
            document.write("bienvenue au DWWM Club");
        }
        maFonction();
        maFonction();
        maFonction();
